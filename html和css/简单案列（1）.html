``` 实现的效果为 ![在这里插入图片描述](https://img-blog.csdnimg.cn/20210306122547350.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3dlaXhpbl81MzA2ODMzOQ==,size_16,color_FFFFFF,t_70) 注：（1）写样式之前一定要写重置样式表 （2）这里可以使用不同种方法去写网页的结构 （3）在ul中，定义了宽高，很明显图片是要比这个尺寸大一点的，故应该用overflow:hidden来裁剪溢出。此时图片就不是一张完整的图片了，可以定义li元素的宽.img-list img{ width: 100%;（高会等比例缩放）使得图片完整显示出来。 （4）在设置图片的margin—bottom是，定义块元素li的样式而不是定义img的样式，我们操作元素一般去操作块元素。 （5）注意三张图片是有一个margin的，我们可以使用 .img-list li{ margin-bottom: 9px;来给三张图片定义一个下外边距，此时第三张图片就会有一个9px的下边距离溢出，不用处理因为其父元素定义了overflow。更好的处理方式是.img-list li:not(:last-child){ margin-bottom: 9px;不给第三张图片定义magin—bottom就好了。 （6）写css是注意层次关系，例如先写body的样式，目的是使得结构层次更清晰，找起来更方便，可维护性高。
